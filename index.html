Space of Questions

Space Of Questions est un jeu de questions ludo-pédagogique basé sur le monde spacial destiné aux élèves de primaire permettant d'apprendre tout en s'amusant.

Le jeu comporte 4 mode de jeux : Facile (10 vies) / Normal (6 vies) / Difficile (3 vies) et Mort Subite (1 vie) en plus du mode Entrainement (Vies infini)
Au démarrage du jeu, l'utilisateur aura la possibilité de se créer un compte permettant de sauvegarder son meilleur score dans chaque mode et de figurer dans le leaderboard.

Après la sélection de la difficulté par l'utilisateur dans le menu, ce dernier répondra à des questions aléatoirement choisies et gagnera 100 points si la réponse est correcte et perdra une vie dans le cas contraire (sans gagner de points, évidemment). Il est bon de mentionner que durant une partie, une fusée sera afficher afin d'illustrer le joueur durant son exploration spaciale. D'autre part, tous les 500 points, une planète apparaitra.

Co-voiturage

Pour ce projet, nous avons choisi l'entreprise de jeux vidéo Blizzard pour réaliser un site web de simulation permettant de faciliter le co-voiturage pour les salariés de cette entreprise. Ce projet a nécessité plusieurs semaines de travail intensif et un apprentissage approfondis des langages à balisage léger HTML et CSS directement pendant le projet. De plus, le projet a été réalisé à deux bien que nous étions 4 à la base.

Ce site n'est pas fonctionnel, car le but du projet était surtout de valider les compétences en Web de première année de BUT (base d'HTML et CSS).

Làmap

Làmap est une application qui permet de calculer le chemin le plus optimisé en fonctions des critères choisis par l'utilisateur qui peut choisir entre les critères suivant : le coût, la consommation de CO2 et/ou le temps du trajet. Une fois le choix des critères choisis par l'utilisateur, un chemin est calculé à l'aide de graphe. Une IHM non-fonctionnelle par manque de temps était demandée pour ce projet, nous avons effectué des prototypes a l'aide de Figma et avons opté pour une IHM orienté mobile bien que cela ne soit pas demandé.

Maqr

Maqr (Maker) est une application développer en python qui permet de créer des QrCode complètement personnalisable. Grâce à cette application, vous pouvez changer la couleur, la taille du qrCode ou encore la taille de ses bordures.

Il est possible de choisir sous quel format on souhaite l'enregistrer le qrCode et il est aussi possible de le visualiser avant de l'enregistrer.

Une interface graphique a été réalisée à l'aide de Tkinter pour ce projet.

Pirattack

Pirattack est un jeu sur terminale réalisé en Java à l'occasion d'un projet pour lequel on a utilisé la méthodologie Agile, nous avons appris à utiliser cette méthodologie et l'appliquer en conditions presque réel.
Nous avons eu le droit à des sprints de 2 heures programmé sur 2 jours et demi pour la réalisation du projet.

Ce jeu s'inspire du jeu Risk, il faut conquérir le plus d'îles pour gagner, avec une possibilité d'acquérir des bonus pour potentiellement renverser la partie.

Ce jeu a été présenté eux étudiants de première année leur permettant de découvrir leurs futurs travaux et pour nous de nous rendre compte des potentiels bugs restant.

KFind

K-Find est un logiciel développé en Java permettant de visualiser des donnée sous la forme d'un nuages de point en utilisant l'algorithme KNN.

Le logiciel s'adapte à tout type de fichier CSV et il est possible d'ajouter un point qui sera classifié en fonction des plus proche voisin (Algorithme KNN).

Ce projet a été découpé en deux parties, la première (Jalon 1) où une première interface graphique était demandée et un logiciel un minimum fonctionnel permettant l'ajout d'un point et une classification sans l'utilisation de l'algorithme KNN.
La seconde partie (Jalon 2) aura, elle était principalement l'implémentation du KNN dans notre logiciel, nécessitant une refonte presque intégrale de notre code.

Une interface graphique a été proposée en utilisant javaFx.
